OK, basicamente este repositorio contiene todo el codigo, pcb y diseño 3D para un robot minisumo de competencias.
La placa fue diseñada con kicad y sus componentes tanto normales como SMD,
un arduino nano, el driver de los motores, las borneras para conectar los mismos y pineras para conectar los sensores infrarrojos como de ultra sonido.
el codigo ya esta listo para correr en un arduino nano con sus pines pmw para los motores y siguiendo todas las reglas del reglamento nacional.
y el diseño 3D esta listo para imprimirse con las medidas exactas 10x10cm, suficiente espacio dentro para colocar los componentes, un techo opcional prar colocar
y un diseño distintivo del pokemon caterpie en pixel art en la parte frontal del mismo
